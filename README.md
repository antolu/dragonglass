# Dragonglass

LLM-powered Obsidian vault agent. Write natural-language prompts in a terminal UI to read and update your vault.

## Setup

```bash
pip install -e ".[dev]"
cp .env.example .env
# Edit .env with your API keys
dragonglass
```

## Usage

Just type naturally:

- `remember that I like cookies`
- `remember that Michael likes flowers`
- `what do I know about Melanie?`
